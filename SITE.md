# Site Documentation: Academy AI - Study Dashboard

**Stitch Project ID:** 7146038398387104347

## 1. Vision & Purpose
A modern, AI-powered study dashboard designed to help users manage summaries, flashcards, news, and research using integrations across the web.

## 2. Target Audience
Students, researchers, and continuous learners who want to centralize their learning process with AI tools.

## 3. Key Capabilities
- View recent summaries from YouTube, articles, PDFs, etc.
- Access an AI Workspace for generating flashcards and quizzes.
- Browse an organized library of previous analyses.
- Manage user settings and profile.
