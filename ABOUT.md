# Sobre o AI Study Hub 🧠🤖

O **AI Study Hub** é mais do que apenas um agregador de notícias; é uma filosofia de aprendizado acelerado aplicada à engenharia de software e Inteligência Artificial.

## 🚀 O Conceito: Estudar IA com IA

Em um mundo onde a informação sobre IA dobra a cada semana, o "ruído" pode ser paralisante. Este projeto nasceu da necessidade de filtrar esse excesso, utilizando a própria tecnologia que estamos estudando para curar o conteúdo.

### Como funciona a colaboração:
1.  **Coleta Automatizada:** Utilizamos **n8n** para monitorar fontes de alta qualidade (Arxiv, blogs de engenharia, papers).
2.  **Processamento com Python:** Scripts customizados realizam a limpeza e estruturação dos dados.
3.  **Refinamento via Antigravity:** Onde o humano entra no loop. Utilizamos o **Google Antigravity** para transformar dados brutos em resumos semânticos e insights acionáveis.

## 🛠️ Filosofia Técnica

*   **Autonomia:** O sistema foi desenhado para ser auto-suficiente na coleta, mas dependente da supervisão humana para a validação final.
*   **Minimalismo:** A interface prioriza o foco. Sem distrações, apenas o conhecimento necessário.
*   **Laboratório Vivo:** Este repositório é constantemente atualizado conforme novas técnicas de Prompt Engineering e Agentes de IA surgem.
